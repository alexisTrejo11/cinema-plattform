# Agents Module

This module contains agent guides and conventions for the Cinema Platform microservices.

## Files

- [AGENTS.md](AGENTS.md) - Main guide for creating new microservices
- [CONVENTIONS.md](CONVENTIONS.md) - Detailed coding conventions and patterns

## Usage

When creating a new service, refer to these documents for:

1. **Project Structure** - Follow the standard directory layout
2. **Architecture** - Apply Hexagonal/DDD patterns
3. **Naming Conventions** - Consistent naming across services
4. **API Design** - REST conventions and patterns
5. **Testing** - pytest setup and testing patterns
6. **Docker** - Containerization guidelines
